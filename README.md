## ðŸ›  Features

- **Text-based multi-agent AI system**  
  - Problem Analyzer: Structures coding problems with inputs, outputs, and constraints

## ðŸ’» Installation (Windows)

1. **Clone the repo**  

```bash
git clone <https://github.com/Chintan1545/-Multi-Agent-AI-Coding-System/tree/main>
cd groq_text_coding
````

2. **Create Conda environment**

```bash
conda create -n groq_text python=3.10 -y
conda activate groq_text
```

3. **Install dependencies**

```bash
pip install streamlit groq python-dotenv
```

4. **Create `.env` file** in project root:

```text
GROQ_API_KEY=your_groq_api_key_here
```
